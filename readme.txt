Searches and replaces the wrong (sedilla) diacritics with the correct ones on data saving

== Description ==

Some people in Romania use wrong diacritics with sedilla. This plugin will replace the wrong letters with correct ones.

== Installation ==

Please follow the [standard installation procedure for WordPress plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).
